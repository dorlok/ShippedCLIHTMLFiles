workflow help

Show help information for a workflow.

Usage
wf help <workflowName>
Argument and Option
Argument and Option Description
workflowName

Required. The name of the workflow. There are two predefined workflow commands currently available:

  • create-and-deploy - Create, bootstrap, and deploy a Shipped project with a single service.
  • fast-deploy - Fast bootstrap and deploy a Shipped project by name.
Command Example
wf help create-and-deploy
